
GEODYN-Kamodo Project Documentation
====================================================

This site contains a collection of information and example Jupyter Notebooks for using the Pygeodyn program.  
Pygeodyn is a wrapper that allows one to read, run, and interpret data from NASA's GEODYN POD software.
